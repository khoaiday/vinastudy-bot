## 🎨 GENERATION TIPS

1. **Consistency:** Characters 1-8 should look like they're from the same game
   - Use approved Lạc Tướng (1) as reference for style
   - Reference previous characters in prompts

2. **Key Features to Verify:**
   - ✅ NOT photorealistic (should look like Kingdom Rush)
   - ✅ Math symbols visible on Hồ Tinh (floating in aura)
   - ✅ All 9 tails visible and numbered on Hồ Tinh
   - ✅ Proper facing direction (male RIGHT, female LEFT)
   - ✅ Vietnamese cultural elements present
   - ✅ Child-appropriate (ages 8-10)

3. **If AI Generates Wrong:**
   - ❌ Photorealistic/cinematic look → Specify "2D stylized, NOT photorealistic"
   - ❌ Chinese aesthetics → Add "distinctly Vietnamese, NOT Chinese"
   - ❌ Math symbols missing → Place description FIRST in prompt
   - ❌ Wrong direction → Explicitly state "facing RIGHT" or "facing LEFT"
   - ❌ Dragons with wings → Specify "wingless rồng Lý dragon"
