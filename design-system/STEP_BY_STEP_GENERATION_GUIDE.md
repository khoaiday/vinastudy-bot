## 🎬 STEP 1: GENERATE LẠC TƯỚNG MALE (STYLE SEED)

This is your most important image. Get this right, then all others follow the same style.

### Step 1.1: Copy the Prompt
1. Open `ALL_PROMPTS_FOR_GENERATION.txt`
2. Find section **#1 LẠC TƯỚNG - MALE**
3. Copy the entire prompt (starting from "2D stylized digital painting..." to "...distinctly Vietnamese aesthetic.")
4. Keep the file name in mind: `lac-tuong-male-avatar.png (1024×1024)`

### Step 1.2: Open GPT Image / DALL-E
1. Go to: https://www.openai.com/dall-e/
2. Sign in with your account
3. You should see a text box that says "A detailed description of the image you'd like"

### Step 1.3: Paste & Generate
1. Click in the text box
2. Paste your copied prompt (Ctrl+V or Cmd+V)
3. Make sure prompt is complete (no truncation)
4. Click **"Generate"** button
5. Wait 10-30 seconds for generation

### Step 1.4: Review the Generated Image

**Check these details:**
- ✅ **Style:** Looks like Kingdom Rush game card (bright, illustrated, 2D)
  - ❌ BAD: Too photorealistic, looks like a movie still
  - ❌ BAD: Looks too anime-like
  - ✅ GOOD: Stylized, bold outlines, bright colors

- ✅ **Appearance:** Vietnamese warrior with red armor
  - ✅ Bronze armor plates visible
  - ✅ Crimson red color prominent
  - ✅ Copper-gold trim visible
  - ✅ Shield visible
  - ✅ Sword visible

- ✅ **Expression:** Friendly confident smile, not stern
  - ✅ Warm, approachable expression
  - ❌ BAD: Too serious/battle-ready
  - ❌ BAD: Looks photorealistic/cinematic

- ✅ **Dragon emblem on chest:** Should look like a long ribbon/eel shape
  - ✅ Long, thin body (snake-like)
  - ✅ Small elegant head
  - ❌ BAD: Chinese-style dragon with wings (too bulky)
  - ❌ BAD: Dragon too thick/wide

- ✅ **Overall vibe:** Game card quality, child-friendly, bright & engaging
  - ✅ Good for ages 8-10
  - ❌ BAD: Too dark/scary
  - ❌ BAD: Too realistic/adult

### Step 1.5: Approval Decision

**If APPROVED ✅:**
→ Go to Step 1.6 (Save image)

**If NOT APPROVED ❌:**
→ Make a note of what's wrong, then:
1. Click **"Try Again"** or **"Generate Again"**
2. OR modify the prompt slightly and retry
3. Common fixes:
   - Add: "NOT photorealistic" if it looks too real
   - Add: "Emphasize 2D stylized, NOT cinematic"
   - Add: "Dragon should be long ribbon-like shape, NOT thick dragon with wings"
   - Try again with revised prompt

### Step 1.6: Download/Save Image

**How to save in GPT Image:**
1. When you're happy with the image, click the image to enlarge it
2. Click **"Save"** or **"Download"** button (depending on interface)
3. Save to your Downloads folder

**Rename the file:**
1. Rename from default name to: `lac-tuong-male-avatar.png`
2. Note size: Must be 1024×1024 (GPT Image should generate this by default)

**Move to correct folder:**
1. Open File Explorer
2. Navigate to: `C:\Users\Nam\OneDrive\Desktop\vinastudy-bot\design-system\characters\playable\lac-tuong\`
3. Move/Copy `lac-tuong-male-avatar.png` here

✅ **STEP 1 COMPLETE!** You now have your style seed.

## 🐛 TROUBLESHOOTING

### Problem: Image looks too photorealistic/cinematic

**Solution:**
- Copy the prompt
- Add at beginning: "NOT photorealistic, NOT hyperrealistic, NOT movie render."
- Regenerate

### Problem: Dragon looks like Chinese dragon (has wings)

**Solution:**
- For Lạc Tướng characters only
- Add to prompt: "Dragon should be RIBBON-LIKE, EXTREMELY LONG and thin (6-8x longer than width), like an eel shape. ABSOLUTELY NO WINGS."
- Regenerate

### Problem: Math symbols not visible on Hồ Tinh

**Solution:**
- Move math symbols description to FIRST of prompt
- Change: "Floating math symbols (+ − × ÷ = √ ½ △ ∑) glowing blue, MUST APPEAR around mystical aura."
- To: "MUST APPEAR: Floating glowing math symbols (+ − × ÷ = √ ½ △ ∑) in neon blue light around mystical aura. PRIORITY: Math symbols must be clearly visible."
- Regenerate

### Problem: Character facing wrong direction

**Solution:**
- Facing is critical for male/female pairs
- Male: "facing slightly RIGHT"
- Female: "facing slightly LEFT" (so they face each other)
- If wrong, regenerate with explicit direction stated

### Problem: Expression too stern/serious (not child-friendly)

**Solution:**
- Change in prompt: "fierce battle-ready" → "friendly confident smile"
- Add: "approachable, warm expression suitable for children ages 8-10"
- Regenerate

### Problem: Doesn't look like Kingdom Rush style

**Solution:**
- Add early in prompt: "similar to Kingdom Rush character art style"
- Add: "game card illustration, NOT anime, NOT cartoon"
- Make sure prompt says "2D stylized digital painting"
- Regenerate

### Problem: Color palette wrong

**Solution:**
- Verify prompt includes correct color hex codes:
  - Lạc Tướng: #C0332E (crimson red) + #C8960C (gold)
  - Đạo Sĩ: #5E2A78 (purple) + #D4960E (gold)
  - Nỏ Thần: #5ABCAA (jade green) + #B87333 (copper)
  - Thần Tướng: #1A3A5C (indigo blue)
- If colors off, regenerate with explicit hex codes in prompt
